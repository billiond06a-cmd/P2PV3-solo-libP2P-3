# P2PV3-solo-libP2P-3

P2P VLAN project with Go core and Flutter/Compose Multiplatform clients.

## Project Structure

- `go_core/`: Go core library for P2P network functionality
- `flutter_app/`: Flutter client application
- `compose-multiplatform-desktop-template/`: Compose Multiplatform client template

## Features

- P2P node initialization and management
- Connection code generation and management
- Peer discovery and connection
- Chat messaging functionality
- Node status monitoring
- Compose Multiplatform support for desktop clients

## Getting Started

### Prerequisites

- Go 1.25.1
- Flutter SDK (for Flutter client)
- Kotlin Multiplatform (for Compose client)

### Building the Go Core

```bash
dev build-go-core
```

### Running the Flutter App

```bash
dev run-flutter-app
```

### Running the Compose Multiplatform App

```bash
dev run-compose-app
```
